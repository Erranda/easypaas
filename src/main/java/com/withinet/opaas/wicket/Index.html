Java, and OSGi in the Cloud

Setting up a Platform-as-a-Service (PaaS) cloud infrastructure is a complex undertaking requiring a significant investment in infrastructure management. EasyPaaS is a light-weight PaaS implementation that allows Java-based systems to be deployed into application containers that are instantiated on-demand and controlled by the infrastructure. These containers are running standard OSGi middleware and can be populated with required services.

About OSGi Cloud

Servlet Containers

Instantiate Web Application services in a matter of seconds, whether for prototyping or running a live enterprise OSGi application.

Control Panel

Evolve your application components at runtime by taking advantage of OSGi lifecycle services.

Provisioning System

If you have a Maven build of your application components, a set of Jar bundles, or a combination of both in a zip file, you can provide these definitions in a Project abstract. Without a single change to your native artifacts these will be deployed into your target containers at runtime.

Collaboration

Invite users to help out a project and configure their permissions as needed to limit access

Multi-tenant Security

Your instances run in a sanboxed environment to prevent malicious activity

Open-source

Built with open source technologies, we provide you different features to help kick start your application.
